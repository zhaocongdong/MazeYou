编写算法：
迷宫是任意的二维数组arraydd：由数字0或1组成，0表示通路，1表示阻塞；

从arraydd的中心（*中心可视为0*）出发，计算走出边界的路径一维数组，可表示为`[(x1, y1), (x2, y2)...]`或`["x1, y1", "x2, y2"...]`

**计算走出迷宫最短路径**

```

0 1 0 1 0 0 0 0 0 0 0
```
